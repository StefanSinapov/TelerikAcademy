## 4. Master Pages

1. Implement a simple Web Forms application using master pages. The application should be like a user profile – separate pages for Personal Info, Friends, Additional Info, etc. + Sidebar for navigation.
2. Implement a Web Forms application using `nested master pages`. It should be a Web site of international company. You should have `master layout page` and few different nested master pages for each country and few content pages in different language, for each country. Define "Home", "About" and "Contacts" pages for each country using its master page.
